# Product Design Resources

## Overview

A curated collection of bookmarks, tools, and articles for digital product designers. This skill covers the research, planning, and resource-gathering phase of product design. For visual execution (code-level UI/UX), use `ui-ux-pro-max`.

The full awesome list is sourced from: [ttt30ga/awesome-product-design](https://github.com/ttt30ga/awesome-product-design)

## When to Use This Skill vs. ui-ux-pro-max

| Phase | Use This Skill | Use ui-ux-pro-max |
|-------|---------------|-------------------|
| Research & Planning | User interviews, personas, JTBD, competitive analysis, empathy maps | — |
| Design Execution | — | Color, typography, layout, components, styles, code |
| Tool Selection | Prototyping tools, accessibility checkers, writing tools | — |
| Asset Sourcing | Free stock photos, videos, illustrations | Use imagegen for AI-generated |
| UX Writing | Readability tools, copy optimization | — |
| Data Sourcing | Open data portals | — |

## Core Categories

### 1. Research Methods

When planning user research, choose from these method categories:

- **Decision Making**: [Untools](https://untools.co) - Thinking frameworks for problem-solving
- **Research Plans**: NN Group, Smashing Magazine guides for UX research planning
- **Stakeholder Interviews**: Templates and checklists from Boxes and Arrows, UX Apprentice
- **User Interviews**: NN Group interviewing guide, UXDesign.cc best practices
- **Personas**: Usability.gov templates, UX Mag foundation guide
- **Empathy Maps**: UX Pin practical guide, Boagworld adaptation tips
- **JTBD (Jobs to be Done)**: Intercom Job Stories, JTBD.info replacing user stories
- **Competitive Analysis**: Xtensio how-to, Hootsuite template
- **Storyboarding**: UX Magazine and UX Planet guides
- **Research Synthesis**: Affinity diagrams, Trello-based synthesis, Tuts+ guide

### 2. Prototyping Tools

External prototyping software (not covered by Codex built-in tools):

- **Quick Mockups**: POP (pen & paper to interactive), Keynotopia (Keynote/PowerPoint prototyping)
- **Web/Mobile Prototyping**: InVision, Marvel, Framer, Proto.io, ProtoPie, Flinto, Origami Studio
- **Animation**: Lottie (Airbnb), Motion UI Design resources list
- **Inspiration**: Hoverstat (alternative digital design), The 12 Principles of Animation

### 3. Stock Assets (Free)

**Stock Photos:**
- ISO Republic, Pexels, All The Free Stock, unDraw (open-source illustrations)

**Stock Videos:**
- Vimeo Free Stock Footage channel

**Visual Tools:**
- Polarr (photo editor), Licecap (animated screen captures), HandBrake (video conversion)

### 4. Accessibility Checkers

External tools to validate accessibility (complementary to ui-ux-pro-max built-in guidelines):

- **WAVE**: Web Accessibility Evaluation Tool
- **Contrast Ratio**: Calculate contrast between any two CSS colors
- **Snook**: Colour Contrast Check
- Key references: Gov.UK accessibility intro, Smashing Magazine color contrast guide

### 5. UX Writing Tools

- **Readability**: Hemingway App, Readable.io, The Writer Readability Checker
- **Trend Analysis**: Google Trends, Google Ngram Viewer
- **Collaboration**: Taskade (collaborative editor and outliner)

### 6. Open Data Sources

For data-driven design and research:

- London Datastore, Data.Gov.UK, NASA Open Data, YouGov, The Tate Collection

## Usage Workflow

1. **Research Phase**: Consult Research Methods category to plan user/stakeholder interviews, create personas, run competitive analysis
2. **Tool Selection**: Pick prototyping tools and accessibility checkers from the list
3. **Asset Gathering**: Source stock photos/videos/illustrations, or use imagegen for AI-generated alternatives
4. **Copy Refinement**: Run UX copy through readability tools
5. **Execution**: Switch to ui-ux-pro-max for visual design and code implementation
